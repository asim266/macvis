# Screenshots

Drop screenshots in this folder named:
- `01-chat.png` — chat with the agent (with a tool call card visible)
- `02-projects.png` — the Projects view
- `03-settings.png` — the Chat API Keys settings tab
- `04-empty-state.png` — the empty chat state with the M logomark

Recommended:
- 1600–2400px wide PNG
- Show the app window with a small amount of background context
- Crop with a 20–40px transparent margin

After saving, the main README will reference them automatically.
